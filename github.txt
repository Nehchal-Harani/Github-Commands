to upload your website code on github

1) open git bash in your code folder

2) git init   (it will create a file in your code)

3) git add .  (it will collect all the files of your code)

4) git commit -m "add your custom messaage" (it will send message to github)

5) git remote add origin https://github.com/Varoon-Vasu/firstwebsite.git

6) git push -u origin master (it will push your code on github)
